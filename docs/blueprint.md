# **App Name**: BaliMedCare

## Core Features:

- Homepage Hero Section: Display clinic name, tagline (Primary Care & 24/7 Homecare in Bali), CTA buttons (Book Appointment, WhatsApp Now).
- Service Overview Cards: Showcase the main services provided with brief descriptions and links to dedicated service pages.
- Doctor Profile Management: Allow admins to add/edit doctor profiles including name, role, short bio, and photo for the 'About Us' page.
- Homecare Request Form: Enable users to submit a request for homecare services with necessary details (address, medical condition, etc.).
- Instagram Feed Integration: Display a preview of the clinic's Instagram feed on the homepage to enhance trust and engagement.
- Smart Location Summarizer: Use a tool to produce an AI-generated summary of available information in order to indicate locations that the clinic serves, by automatically pulling them from submitted Homecare Request Forms

## Style Guidelines:

- Primary color: Deep teal (#008080) to convey a sense of health, reliability, and trust, suitable for a medical brand.
- Background color: Very light greenish-blue (#E0F8F7), offering a calm and clean feel.
- Accent color: Pale greenish-yellow (#90EE90) to highlight calls to action, representing healing and safety.
- Body and headline font: 'PT Sans', a humanist sans-serif providing a modern yet accessible and friendly feel, well-suited for medical content.
- Use clean, modern icons with rounded edges to represent services and key highlights.
- Employ a clean and spacious layout with clear sections and consistent design patterns throughout the site.
- Incorporate subtle animations, such as smooth scrolling and fade-in effects, to enhance user engagement.
